# Калькулятор

Браузерный калькулятор с современным дизайном и поддержкой клавиатуры.

## Возможности

- Базовые математические операции (+, -, *, /)
- Поддержка десятичных чисел
- Поддержка клавиатуры
- Адаптивный дизайн
- Современный UI

## Деплой на Vercel

Проект подготовлен для деплоя на Vercel. Вы можете задеплоить его следующими способами:

### Способ 1: Через Vercel CLI

1. Установите Vercel CLI (если еще не установлен):
```powershell
npm install -g vercel
```

2. Войдите в Vercel:
```powershell
vercel login
```

3. Задеплойте проект:
```powershell
vercel
```

4. Для продакшн деплоя:
```powershell
vercel --prod
```

### Способ 2: Через GitHub

1. Создайте репозиторий на GitHub и запушьте код
2. Зайдите на [vercel.com](https://vercel.com)
3. Войдите через GitHub
4. Нажмите "Add New Project"
5. Выберите ваш репозиторий
6. Vercel автоматически определит настройки проекта
7. Нажмите "Deploy"

### Способ 3: Через веб-интерфейс Vercel

1. Зайдите на [vercel.com](https://vercel.com)
2. Перетащите папку проекта в область загрузки
3. Vercel автоматически задеплоит проект

## Структура проекта

- `index.html` - главная страница калькулятора
- `package.json` - конфигурация проекта
- `vercel.json` - конфигурация для Vercel
